6.1. v0.2.0, FIXME¶

  • Switch to containers rather than virtual machines. (VM code remains available under the virtual-machines branch.)

6.2. v0.1.5, 2015-Jun-16¶

  • First public release.

6.3. v0.1.4, 2015-Jun-04¶

  • Stop using VDE in workstation mode; use OS bridge/NAT instead (#40).
  • Various bug fixes.

6.4. v0.1.3, 2015-Apr-30¶

  • Remove chextern secondary job user (#51).
  • Add --version argument to scripts (#54).
  • Pass tests under QEMU 2.3.0.
  • Various bug fixes.

6.5. v0.1.2, 2015-Feb-09¶

  • Regression testing.
  • Performance improvements (virtio-blk data plane driver, fallocate() for temp block devices).
  • Better error handling.
  • API updates.
  • Various bug fixes.

  • Write documentation.
  • API updates.
  • Various bug fixes.
